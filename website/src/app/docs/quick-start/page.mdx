# Quick Start

Get your next big project started in 60 seconds with Trix.

## 1. Prerequisites

Before using Trix, ensure you have the following installed:
- [Node.js](https://nodejs.org) (v18.0.0 or higher)
- A package manager (`npm`, `yarn`, `pnpm`, or `bun`)

## 2. Installation

You can run Trix directly using `npx`, or install it globally for faster access.

```bash
# Using npx (Recommended for one-time use)
npx create-trix@latest my-app

# Global Installation
npm install -g create-trix
```

## 3. Create your first project

Once installed, simply run the command and follow the interactive prompts to select your stack.

```bash
create-trix
```

### Prompt Options

- **Project Name**: Choose a name for your new directory.
- **Type**: Frontend, Backend, or Fullstack (Coming soon).
- **Framework**: Select from React, Vue, Svelte, or Node.js frameworks.
- **Customization**: Add Tailwind, Auth, Databases, and more.

## 4. Launch!

Navigate to your new project and start developing.

```bash
cd my-app
npm run dev
```
